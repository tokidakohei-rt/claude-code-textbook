# Chapter 0: はじめに

## この講座について

この講座では、2時間でClaude Codeの応用機能を理解し、最終的に **指定した業界の研究レポート** を作ります。

### Vol.1（前回）との比較

| | Vol.1 | Vol.2（今回） |
|---|---|---|
| 入力 | 手元のPDF | Webから取得する最新情報 |
| 出力 | スライド | 業界研究レポート |
| 学ぶ機能 | Rules, Skills, Subagents | Web検索, Hooks, パイプラインSkills, Agent Teams |

Vol.1 が「手元の情報を整理する力」だとすると、Vol.2 は **「情報を集めて分析する力」** です。

### 今日のゴール

- Web検索を使った情報収集ができる
- Hooks で自動化トリガーを設定できる
- 複数の Skills を繋げたパイプラインを設計できる
- Agent Teams で並列調査を指揮できる
- 業界研究レポートを完成させる

### 前提条件

- Claude Code の基本操作ができること（起動、ファイル操作、Skills の概念）
- Vol.1 修了済み、または上記の基礎知識があること
- プログラミング経験は不要

## レビュアーエージェント

この教材には **`/review`** というSkillが最初から入っています。

試しに、今すぐ実行してみましょう:

```
/review README.md
```

いい点と改善点をフィードバックしてくれるはずです。この講座では、何かを作るたびに `/review` を使ってフィードバックをもらい、改善するサイクルを繰り返します。

> **ポイント**: このレビュアーの正体は、Chapter 7 で種明かしします。まずは「使う側」として体験してください。

## 講座の流れ

```
Phase 1（20分）: 復習 → セットアップ
     ↓
Phase 2（40分）: Web検索 → Hooks
     ↓
Phase 3（30分）: パイプラインSkills → Agent Teams
     ↓
Phase 4（30分）: 業界調査 → レポート生成 → レビュー
```
