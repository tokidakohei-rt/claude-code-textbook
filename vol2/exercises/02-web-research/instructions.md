# 演習 2: Web検索と情報収集

## 課題

### Task 1: 基本的なWeb検索

Claude Code に以下を頼んでみてください:

```
「リモートワーク 普及率 2024年 日本」について検索して
```

検索結果を確認し、以下を書き出してください:
- 見つかった具体的な数字（普及率 ○○%など）
- 情報源はどこか

### Task 2: 複数キーワードで検索の精度を比較

以下の2つの検索を試して、結果の違いを比べてください:

1. 「AI市場」で検索
2. 「日本 生成AI市場 市場規模 2024年 調査レポート」で検索

どちらがより有用な情報を返しましたか？なぜそうなるか考えてみましょう。

### Task 3: 情報の裏取り

Task 1 または Task 2 で見つかった数字について、別の情報源でも同じ数字が確認できるか調べてください。

```
[見つかった数字]について、別の情報源でも確認してください
```

### Task 4: `/research` Skill を実行する（チャレンジ）

好きなテーマを1つ選んで `/research` を実行してみてください:

```
/research [テーマ]
```

出力されたファイルを `/review` でレビューし、フィードバックを確認しましょう:

```
/review output/research-[テーマ].md
```

## 確認ポイント

- [ ] Web検索で具体的な数字を取得できた
- [ ] 検索キーワードの具体性が結果に影響することを理解した
- [ ] 情報の裏取りを体験した
- [ ] （チャレンジ）/research を実行し、/review でフィードバックを受けた

完了したら「演習2終わりました」と伝えてください。
