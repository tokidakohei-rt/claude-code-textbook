# 演習 1: 復習と環境確認

## 課題

### Task 1: 基本ツールの確認

以下の操作を Claude Code で実行してください:

1. `curriculum/` フォルダ内のファイル一覧を表示する
2. `curriculum/00-introduction.md` の内容を読む
3. `output/` フォルダに `test.txt` というファイルを作り、「Hello Vol.2」と書き込む
4. 作成した `test.txt` を削除する

### Task 2: Skills の確認

`.claude/commands/` フォルダ内にある Skill ファイルをすべて読み、それぞれ何をするものか一言で説明してください。

### Task 3: レビュアーを体験する

`/review` を使って `curriculum/00-introduction.md` をレビューしてみてください。

```
/review curriculum/00-introduction.md
```

フィードバックの内容を読んで、以下を考えてみましょう:
- どんな観点でレビューされたか
- フィードバックは的確だったか

### Task 4: CLAUDE.md を読む

このリポジトリの `CLAUDE.md` を読んで、Vol.1 との違いを3つ見つけてください。
