# Claude Code Textbook — 2時間で論文要約スライドを作る

Anthropic 公式 CLI ツール「Claude Code」を使って、論文を読み解きスライドにまとめるまでを体験する実践型の教材です。非エンジニアでも、AI エージェントの基本操作から Skills・Subagents といった応用概念までを約2時間で学べます。

## はじめに

1. **[セットアップガイド（SETUP.md）](./SETUP.md)** を読み、必要なツールをインストールしてください
2. セットアップが完了したら、リポジトリのフォルダで `claude` を起動し「始めよう」と話しかけてください

## カリキュラム構成

| フェーズ | 時間目安 | 内容 |
|---------|---------|------|
| Phase 1: 基礎 | 30分 | Claude Code の基本操作、ファイル読み書き |
| Phase 2: 設計 | 30分 | Rules, Skills, Subagents |
| Phase 3: 実践 | 30分 | 論文を読み解く |
| Phase 4: 成果物 | 30分 | スライド作成・仕上げ |
