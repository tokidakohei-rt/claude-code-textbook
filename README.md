# Claude Code Textbook

Anthropic 公式 CLI ツール「Claude Code」を実践的に学ぶ教材シリーズです。非エンジニアでも、AI エージェントの基本操作から応用までを体験しながら習得できます。

## シリーズ構成

| Vol. | タイトル | 時間 | 学ぶ機能 |
|------|---------|------|---------|
| [Vol.1](./vol1/) | 2時間で論文要約スライドを作る | 2時間 | 基本操作, Rules, Skills, Subagents |
| [Vol.2](./vol2/) | 2時間で業界研究レポートを作る | 2時間 | Web検索, Hooks, パイプラインSkills, Agent Teams |

## はじめ方

### Vol.1 から始める場合（Claude Code が初めての方）

```bash
cd claude-code-textbook/vol1
claude
# → 「始めよう」と話しかけてください
```

### Vol.2 から始める場合（Claude Code の基本操作がわかる方）

```bash
cd claude-code-textbook/vol2
claude
# → 「始めよう」と話しかけてください
```
