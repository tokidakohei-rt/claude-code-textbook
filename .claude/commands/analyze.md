$ARGUMENTS で指定されたファイル（調査結果）を読み込み、分析を行ってください。

## 分析の観点

1. **市場の構造**: 誰がどのポジションにいるか。勢力図はどうなっているか
2. **成長ドライバー**: 何がこの市場を伸ばしているか（3つ以内）
3. **リスク要因**: 成長を阻害しうる要因は何か（3つ以内）
4. **機会**: 今後有望な領域やアプローチは何か
5. **So What?（だから何？）**: この業界について最も重要な示唆は何か

## 分析のルール

- 調査結果に書かれた事実をベースにする（憶測を足さない）
- 数字同士を比較・関連づける（「A社のシェアはB社の2倍」等）
- 「だから何？」を必ず言語化する
- 箇条書きは各項目3つ以内に絞る

## 出力フォーマット

```markdown
# 分析結果: [テーマ]

## エグゼクティブサマリー
[3行以内で要点をまとめる]

## 市場構造
- ...

## 成長ドライバー
1. ...
2. ...
3. ...

## リスク要因
1. ...
2. ...
3. ...

## 今後の機会
- ...

## So What?
> [1〜2文で最も重要な示唆を記述]
```

分析結果は `output/analysis-[テーマ名].md` に保存してください。
