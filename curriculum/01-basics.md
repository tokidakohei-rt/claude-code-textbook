# Chapter 1: 基本操作 - 対話とファイル操作

## Claude Codeの対話モデル

Claude Codeは **チャット形式** でやり取りします。
あなたが自然言語で指示を出すと、Claudeがツール（Read, Edit, Write, Bash等）を使って作業します。

### ツールの承認モデル

Claude Codeがツールを使うとき、権限が必要です:

| モード | 説明 |
|-------|------|
| 毎回確認 | すべてのツール実行前に確認を求める |
| 自動許可 | 安全な操作は自動実行（デフォルト推奨） |

ファイルの読み取りは基本的に安全なので自動許可されます。
ファイルの書き込みやコマンド実行は確認が求められます。

## 主要ツール

### 1. Read - ファイル読み取り
```
「このファイルを読んで」
「package.json の中身を見せて」
```
→ Claude Codeは `Read` ツールを使ってファイルの内容を表示します。

### 2. Edit - ファイル編集
```
「3行目の "hello" を "world" に変えて」
「関数名を camelCase に修正して」
```
→ `Edit` ツールで既存ファイルの一部を書き換えます。

### 3. Write - ファイル作成
```
「新しいファイル test.txt を作って」
```
→ `Write` ツールでファイルを新規作成します。

### 4. Bash - コマンド実行
```
「git status を実行して」
「npm install して」
```
→ `Bash` ツールでシェルコマンドを実行します。

### 5. Glob / Grep - 検索
```
「.md ファイルを全部探して」      → Glob（ファイル名パターン検索）
「TODO という文字列を探して」     → Grep（ファイル内容検索）
```

## ポイント: 自然言語で指示する

Claude Codeでは、プログラミング的な記法ではなく **普通の日本語** で指示できます。
「あのファイルのあの部分を直して」のような曖昧な指示でも、コンテキストから推測して動きます。
