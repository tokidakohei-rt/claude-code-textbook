# Chapter 0: はじめに

## この講座について

この講座では、2時間でClaude Codeの全体像を理解し、最終的に **論文の要約スライド** を作ります。

### 今日のゴール
- Claude Codeの主要機能を一通り体験する
- Rules, Skills, Subagents の仕組みを理解する
- 実際に論文を読み、要約スライドを生成する

### 前提条件
- Claude Codeがインストール済みであること
- ターミナルの基本操作ができること
- プログラミング経験は不要

## Claude Code とは？

Claude Codeは、Anthropicが提供する **CLIベースのAIアシスタント** です。

特徴:
- ターミナル上で動作する
- ファイルの読み書き、コマンド実行ができる
- プロジェクトのコンテキスト（CLAUDE.md等）を理解して動作する
- Subagent（子エージェント）を使った並列処理ができる

### IDE版との違い

VS Code拡張やJetBrains拡張もありますが、本講座ではCLI版を前提とします。
（IDE版でも基本概念は同じです）

## 基本コマンド一覧

| コマンド | 説明 |
|---------|------|
| `claude` | Claude Codeを起動 |
| `/help` | ヘルプを表示 |
| `/clear` | 会話履歴をクリア |
| `/compact` | 会話を要約して圧縮 |
| `/cost` | 現在のセッションのコストを表示 |
| `/status` | 現在の設定状態を表示 |
| Esc×2 | 実行中の操作をキャンセル |
| Ctrl+C | Claude Codeを終了 |
