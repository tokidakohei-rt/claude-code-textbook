```json
{
  "languages": [
    {"name": "Python", "type": "動的型付け", "year": 1991},
    {"name": "TypeScript", "type": "静的型付け", "year": 2012},
    {"name": "Rust", "type": "静的型付け", "year": 2010}
  ]
}
```

### Task 3: データの編集
作成した `languages.json` に、以下の言語を追加してください:
- Go（静的型付け、2009年）

**条件**: ファイル全体を書き直すのではなく、Edit（部分編集）で追加してください。
